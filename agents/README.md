# Agents Directory

This directory defines startup instructions per agent.

Read order:
1. `../ops/START_HERE.md`
2. `common/FIRST_READ.md`
3. your own file (`codex/`, `gemini/`, `claude/`)

Note:
- Gemini uses `gemini/BOOTSTRAP.md` as a single-entry startup file.
- Codex uses `codex/BOOTSTRAP.md` as a single-entry startup file.
- Claude uses `claude/BOOTSTRAP.md` as a single-entry startup file.
