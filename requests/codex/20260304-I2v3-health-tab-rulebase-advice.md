# Request: からだタブ ルールベースアドバイス（I2v3）

- Date: 2026-03-04
- Owner: Codex
- Status: `todo`
- Phase: I（タブ強化）
- Depends on: I2v2（完了済み）

## Background

からだタブには現在 `TabAiAdvice` コンポーネント（LLM生成コメント）が一番下に配置されている。
問題:
1. LLM生成コメントは「condition」一種類で、体重/血圧/睡眠の区別がない
2. レポート未生成日はコメントが表示されない
3. 各内タブのデータに即したフィードバックがない

## 変更内容

### 1. `TabAiAdvice` を廃止し、ルールベースアドバイスカードを最上段に配置

各内タブコンポーネント（CompositionTab, CirculationTab, SleepTab）の**最上段**（`health-current-card` の上）に、データから自動生成するアドバイスカードを表示する。

#### UIデザイン

```
┌──────────────────────────────────────┐
│ 💡 アドバイステキスト                     │
└──────────────────────────────────────┘
```

- アバター不要、シンプルなテキストカード
- CSSクラス: `health-advice-card`（新規）
- 背景: `var(--bg-card)` + 左ボーダー 3px `var(--accent-indigo)`
- パディング: 12px 16px
- フォント: 13px, color: `var(--text-secondary)`
- アイコン: 💡（テキスト絵文字）を先頭に
- データがない（コメント生成不可）場合はカード自体を非表示

### 2. ルールベースロジック（フロントエンドのみ）

各タブコンポーネント内で、読み込み済みのデータから以下のルールでコメントを生成する。
`generateAdvice(data, segment)` のような純粋関数を各タブの上部に定義する。

#### 体重タブ (CompositionTab)

| 条件 | コメント |
|------|---------|
| BMI < 18.5 | 「BMIが低めです。バランスの良い食事を心がけましょう。」 |
| 18.5 ≤ BMI < 25 | 「BMIは標準範囲内です。この調子を維持しましょう。」 |
| 25 ≤ BMI < 30 | 「BMIがやや高めです。食事と運動のバランスを見直してみましょう。」 |
| BMI ≥ 30 | 「BMIが高めです。かかりつけ医への相談もご検討ください。」 |
| 週表示 & 体重減少 > 0.5kg | 「今週は{diff}kg減少しました。順調です。」（BMIコメントに追加） |
| 週表示 & 体重増加 > 1.0kg | 「今週は{diff}kg増加しています。食事内容を振り返ってみましょう。」（BMIコメントに追加） |
| 目標体重あり & 差 ≤ 2kg | 「目標体重まであと{diff}kgです。もう少しです！」（追加） |
| データなし (BMI null) | カード非表示 |

複数条件に該当する場合は「。」区切りで結合する。

#### 血圧・心拍タブ (CirculationTab)

| 条件 | コメント |
|------|---------|
| 正常 (sys < 125 & dia < 75) | 「血圧は正常範囲です。引き続き健康的な生活を。」 |
| 高値血圧 (sys 125-134 or dia 75-84) | 「血圧がやや高めです。塩分の摂りすぎに注意しましょう。」 |
| I度高血圧 (sys 135-144 or dia 85-89) | 「血圧が高い状態が続いています。生活習慣の改善を検討しましょう。」 |
| II度以上 (sys ≥ 145 or dia ≥ 90) | 「血圧が高い状態です。かかりつけ医への相談をおすすめします。」 |
| 安静時心拍 < 60 | 「安静時心拍が低めです。体調に変化がないか確認しましょう。」（追加） |
| 安静時心拍 ≥ 80 | 「安静時心拍がやや高めです。リラックスする時間を作りましょう。」（追加） |
| 高血圧判定日 ≥ 3 (期間表示) | 「期間中に高血圧判定が{n}日ありました。経過を注視しましょう。」（追加） |
| データなし (BP & HR null) | カード非表示 |

#### 睡眠タブ (SleepTab)

| 条件 | コメント |
|------|---------|
| 睡眠 ≥ 420分 | 「十分な睡眠が取れています。この調子を維持しましょう。」 |
| 360 ≤ 睡眠 < 420分 | 「睡眠がやや短めです。もう少し早めの就寝を心がけてみましょう。」 |
| 睡眠 < 360分 | 「睡眠時間が不足しています。睡眠の質と量の改善を意識しましょう。」 |
| 深い睡眠比率 < 15% | 「深い睡眠の割合が少なめです。就寝前のスマホ利用を控えると改善されることがあります。」（追加） |
| SpO2 < 95% (avg) | 「血中酸素濃度が低めです。気になる場合は医師に相談してください。」（追加） |
| データなし | カード非表示 |

## 制約

1. 各タブコンポーネント内でデータから直接コメント生成（API追加不要）
2. コメント生成は純粋関数で実装（テスト可能に）
3. TypeScript ビルドが通ること
4. 既存のチャート・データ表示は変更しない
5. 本番デプロイはしない（ローカル確認のみ）

## Acceptance Criteria

1. 各内タブの最上段にルールベースのアドバイスカードが表示される
2. コメント内容がデータに即している（BMI, 血圧分類, 睡眠時間に応じて変化）
3. データがない場合はカード非表示
4. 既存の `TabAiAdvice`（LLMコメント）が削除されている
5. TypeScript ビルドが通る
6. 本番デプロイはしない
