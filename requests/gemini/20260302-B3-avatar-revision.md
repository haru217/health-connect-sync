## 修正理由
初版のアバター画像をレビューした結果、仕様との乖離と表示サイズの考慮不足が判明しました。

## 必須修正

### 1. 構図をフェイスショットに変更（全3枚共通）
アプリ内の表示サイズは30〜48pxの小さな円形です。バストアップでは顔が小さすぎて表情が判別できません。

**変更**: 肩から上のクローズアップ。顔が画像面積の70%以上を占める構図にする。
**参考**: LINE・Slackのプロフィール画像のような、顔がしっかり見える比率。

### 2. ユウ先生 — 白衣と聴診器を外す
仕様では「白衣は着ない（カジュアルだが清潔感のある服装）」と指定しています。
→ カジュアルなシャツ（例: ポロシャツ、ボタンシャツ）に変更。聴診器も外す。

### 3. サキさん — エプロンと英語テキストを外す
仕様では「エプロンや食材は持たない（シンプルなポートレート）」と指定しています。
→ シンプルなブラウスに変更。「REGISTERED DIETITIAN」テキストを削除。

### 4. マイコーチ — 構図を他2枚と揃える
現在は横長で全身寄りの構図です。他2枚と同じ正方形・フェイスショットに統一する。
サムズアップポーズは不要（顔クローズアップなので手は映らない）。

### 5. 背景を透明にする
3枚とも白背景になっています。仕様どおり透明背景（PNG透過）にする。

### 6. 画風を統一する
現在3枚の画風がバラバラです（繊細なタッチ、柔らかいタッチ、太い線のタッチ）。
同じ画風・同じ線の太さ・同じ塗りの質感で統一してください。
仕様の「フラットイラスト / セミリアル」に寄せる方が望ましいですが、アニメ調で統一するなら3枚とも同じトーンで。

## 推奨修正

### 7. ファイル名を仕様に合わせる
```
avatar-yu.png
avatar-saki.png
avatar-mai.png
```
配置先: `web-app/public/avatars/`

## サイズ確認用の参考情報
```
実際の表示: 30x30px〜48x48pxの円形（CSSで丸くクロップ）
納品サイズ: 512x512px（正方形）
→ 10倍以上縮小されるため、細かいディテールより「顔の印象」が重要
```

## Acceptance Criteria
1. 3枚ともフェイスショット（顔70%以上）で統一
2. ユウ先生に白衣・聴診器がない
3. サキさんにエプロン・英語テキストがない
4. 3枚の画風が統一されている
5. 背景が透明
6. 512x512px以上の正方形PNG
7. 30px円形に縮小しても3人の区別がつく
